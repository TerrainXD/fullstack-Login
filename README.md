# Installation & Setup
```sh
git clone https://github.com/TerrainXD/fullstack-Login.git

cd fullstack-Login
```

# Setup Backend
```sh
cd backend

npm install

node server.js

```

# Setup Frontend
```sh
cd frontend

npm install

npm start

```
